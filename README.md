# Landing Page Project

This is a **Landing Page project** completed as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.  
The assignment was to practice **HTML and CSS fundamentals**, especially **Flexbox**, by building a simple page layout.

---

## 📚 What This Project Covers
- Structuring a webpage with semantic HTML  
- Styling with CSS (colors, fonts, spacing)  
- Using **Flexbox** for layout and alignment  
- Creating a basic landing page design  

---

## 🛠️ Technologies Used
- **HTML5**  
- **CSS3**  
- **Flexbox**
